# Lily
Dark theme for Chromium

## About theme
I made it after I found old 2017 theme on [Chrome Web Store](https://chromewebstore.google.com/detail/awesome-anime-white-editi/fnomgkakoaflafmeonlomogchidbmadp).
But I liked wallpapers and wanted to change colours a bit, so created my own one.

## How to install
*P.S. As I use [ungoogled-chromium](https://github.com/ungoogled-software/ungoogled-chromium) I wrote process for installation based on it.*
- Download [*.zip* archive](https://github.com/TechAngle/Lily/raw/refs/heads/main/Lily.zip).
- Go to `chrome://extensions/` and **turn on** `Developer mode`
- Drag-and-drop *.zip*.
- **Done**.
